# 📌 Problem Statement

Preparing an SME IPO offer document is currently a manual, time-consuming, and expensive process.

SME promoters depend heavily on merchant bankers, legal experts, and compliance professionals to prepare disclosure-ready documents that comply with SEBI regulations.

This project aims to reduce preparation time, improve document quality, and lower drafting costs while keeping authorized intermediaries involved in the final review process.

IPO Pilot AI - https://ipo-pilot-ai.vercel.app

# 💡 Solution

<p align="center">
  <img src="screenshots/Screenshot%202026-07-06%20215826.png" width="100%">

IPO Pilot AI assists SME founders by generating a structured draft IPO offer document using AI.

The platform combines:

- 📄 Intelligent document upload
- 🔍 OCR-based information extraction
- 🤖 AI-assisted drafting
- ✅ SEBI disclosure validation
- ⚠️ Gap & inconsistency detection
- 👨‍⚖️ Human review workflow
- 📥 Export-ready documentation

# 🔒 Responsible AI

IPO Pilot AI follows a Human-in-the-Loop approach.

- AI generates draft content
- Human experts validate every disclosure
- No automatic regulatory submission
- Transparent confidence scoring
- Regulatory knowledge base can be updated
